# Fitness Coaching Website

A modern, responsive website for a fitness coaching business built with HTML, CSS, and JavaScript. The site uses Bootstrap 5 for the layout and components, along with custom styling and animations.

## Features

- Responsive design that works on all devices
- Modern and clean user interface
- Smooth scrolling and animations
- Contact form with validation
- Interactive navigation
- Optimized performance

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Bootstrap 5
- Font Awesome Icons
- Google Fonts (Poppins)

## Setup

1. Clone the repository
2. Open `index.html` in your web browser
3. No build process or dependencies required

## Customization

### Colors
The color scheme can be modified in the `css/style.css` file by changing the CSS variables in the `:root` selector:

```css
:root {
    --primary-color: #1a1a1a;
    --secondary-color: #ff4d4d;
    --text-color: #333;
    --light-bg: #f8f9fa;
    --white: #ffffff;
}
```

### Images
Replace the images in the `images/` directory with your own:
- `hero-image.jpeg`: Main hero section image
- `about-image.jpg`: About section image
- `pattern.png`: Background pattern for the hero section

### Content
Edit the content in `index.html` to match your business information and services.
